# Smart Photo Retouching with Interactive Doodle Overlays

```text
# 智能修图 + 涂鸦互动叠加｜创意照片成图提示词

## 1. 任务目标

基于我上传的原图，生成一张“**智能修图 + 涂鸦互动叠加**”的创意照片成图。

先对原图进行适度且自然的优化，让画面在**光线、色彩、氛围、质感、清晰度、视觉重心**等方面更适合后续创作；再在此基础上叠加具有设计感、互动感和节奏感的涂鸦元素、文字元素及装饰元素，使画面既保留真实照片的生活感，又具备潮流感、趣味感、传播感与可直接发布的完成度。

若用户没有特别说明，则默认开启“智能修图”；若用户明确要求关闭修图，则保留原图质感，仅进行创意叠加；若用户指定修图风格，则优先按用户要求执行。

---

## 2. 总体要求

### 2.1 核心原则

你需要先**理解照片内容**，再决定如何创作，而不是套用固定模板。

请智能识别并综合判断原图中的以下信息：

- 人物 / 主体是谁
- 动作与姿态
- 视线方向
- 表情与情绪
- 场景类型
- 光线状态
- 色彩环境
- 画面节奏与主次关系
- 整体气质与适合传播的视觉方向

然后自动匹配最适合这张图的：

- 修图方向
- 涂鸦方式
- 文字内容
- 文字语言
- 字体风格
- 色彩倾向
- 互动元素
- 装饰节奏
- 画面密度
- 风格完成度

这些内容都不应固定，而应根据原图自动变化。若用户有明确要求，则优先按用户要求；若没有，则由系统智能判断最优方向。

---

## 3. 成图效果定位

最终效果应当是：

> 原始照片经过自然、适度、审美在线的修图优化后，再被创意化、风格化、互动化地升级，形成一张适合发朋友圈、社交平台、小红书、微博、头像分享、纪念记录或轻海报用途的高完成度创意图片。

风格关键词可以根据内容自由变化，但整体应满足以下要求：

- 真实感仍在，不要把照片完全做成失真插画
- 创意感明显，不是普通滤镜图
- 涂鸦元素不是简单贴图，而是“参与画面”
- 杂乱但有秩序，丰富但不脏乱
- 好看、自然、适合公开发布
- 具有一定“让人愿意停留观看”的视觉吸引力

---

## 4. 执行流程

### 4.1 第一步：智能修图

默认对原图进行适度优化，包括但不限于：

- 调整曝光、对比度、白平衡、明暗层次
- 优化色彩统一性与氛围感
- 提升主体清晰度与视觉重心
- 优化肤色、面部状态、画面质感
- 减弱噪点、杂色、凌乱背景带来的干扰
- 让整体画面更适合后续叠加创作

注意：

- 修图必须自然，不可过度磨皮、过度锐化、过度假白
- 人像类需保留人物辨识度、五官特征与真实气质
- 风景类需保留环境氛围和空间层次
- 美食类需突出食物质感与诱人程度，但避免塑料感
- 日常照片需保持“真实生活感”，不能修得像廉价影楼图

若用户明确要求某种修图风格，则按指定风格执行，例如：

- 清透自然
- 胶片感
- 复古暖调
- 日系淡彩
- 电影感
- 港风
- 高级冷淡
- 明快活泼
- 夜景增强
- 食欲提升
- 旅行氛围感
- 纪实街拍感
- 其他用户指定方向

---

### 4.2 第二步：智能叠加创意涂鸦

在修图完成后，叠加具有设计感和互动感的视觉元素，包括但不限于：

- 手绘线条
- 涂鸦笔触
- 文字标题 / 短句 / 拟声词
- 小角色 / 小生物 / 小人 / 小图标
- 喷绘色块
- 笔刷痕迹
- 贴纸感元素
- 光环、箭头、动作线、星点、云团、边框
- 小物件、小符号、小型视觉语言
- 角落装饰、框选元素、互动连线

所有叠加内容都应与原图构成**自然互动关系**，而不是生硬堆砌。

---

### 4.3 第三步：处理画面中不够好看或不宜直接公开的区域

请智能识别原图中以下可能需要处理的区域，并以自然、巧妙、风格统一的方式进行美化、弱化、遮挡或转化：

- 杂乱背景
- 空白死角
- 视觉干扰区域
- 不够美观的边角
- 不宜公开展示的私密细节
- 可能暴露个人信息的内容
- 凌乱桌面、路人、杂物、反光、多余文字
- 影响整体观感的瑕疵区域

处理方式可包括但不限于：

- 小贴纸
- 手绘涂鸦遮挡
- 色块覆盖
- 角标元素
- 小图形
- 小角色
- 局部模糊弱化
- 风格化笔刷融合
- 创意性装饰性改写

要求处理结果看起来像是**设计的一部分**，而不是为了遮挡而遮挡。

---

## 5. 按图片类型智能分流处理

请优先判断原图属于哪一类，再决定创作策略。图片类型主要参考日常社交平台、朋友圈常见内容。

### 5.1 人像 / 自拍 / 半身照 / 他拍照

重点处理方向：

- 保留人物辨识度与真实神态
- 强调人物气质、动作、视线与情绪
- 涂鸦元素应围绕人物产生互动，如环绕、托举、牵引、注视、陪伴、吐槽、强调动作
- 可加入适合人物风格的小角色、小生物、表情符号、光环、动作线、个性短句
- 若是单人照，可突出“主角感”
- 若是情侣 / 朋友合照，可强化关系感、互动感、回忆感

适合的风格方向可根据画面自动选择：

- 少女感 / 少年感
- 可爱童趣
- 酷感街头
- 轻复古
- 高级杂志感
- 松弛感日常
- 校园清新
- 温柔纪念感
- 潮流拍立得拼贴感

---

### 5.2 多人合照 / 聚会 / 毕业 / 团建 / 朋友局

重点处理方向：

- 统一整体氛围与色彩
- 突出“关系感”和“共同记忆”
- 可为不同人物加不同的小互动标签或装饰
- 文字更适合做成轻松、群体感、纪念感的短句
- 装饰节奏要均衡，避免只堆在一个角落

可加入：

- 群体箭头标注
- 名字缩写 / 小标签
- 欢乐拟声词
- 纪念日期感小元素
- 星星、彩带、手写边注、相机框线等

---

### 5.3 儿童 / 亲子 / 宠物类照片

重点处理方向：

- 更童趣、活泼、跳跃、柔软
- 色彩可更明亮、清新、圆润
- 适合加入互动型小生物、小玩具、小云朵、小太阳、蜡笔感元素
- 字体更可爱、圆润、手写化

若是宠物图：

- 强化宠物表情和动作的“拟人化趣味”
- 涂鸦可以围绕宠物性格展开，如傲娇、呆萌、开心、巡视、躺平等
- 小物件可与宠物互动，如鱼骨、毛线球、小皇冠、爪印、爱心等

---

### 5.4 美食 / 饮品 / 探店 / 甜品图

重点处理方向：

- 优先突出“食物本身”的色泽、质感、层次与食欲感
- 涂鸦不能压过主体
- 装饰更适合轻巧、精致、有节制
- 可适度加入餐具线稿、热气线、评价短句、口感关键词、手写标签

文字方向可偏向：

- 好吃、治愈、今日份快乐
- 某种口感 / 心情 / 小情绪
- 一两句简洁、有点俏皮但不过度网感的表达

风格方向可偏向：

- 日常美食记录
- 探店感
- 杂志菜单感
- 日式轻手账
- 法式甜点感
- 轻复古餐桌感

---

### 5.5 风景 / 旅行 / 城市街景 / 海边 / 山水 / 日落图

重点处理方向：

- 保留环境空间感与氛围感
- 涂鸦应辅助表达“风的方向、光的感觉、旅行的情绪、时间的痕迹”
- 可加入路线感箭头、地名感小标签、天气感元素、星轨感、云朵、小飞鸟、旅行短句等
- 文字不宜过多，应留白适度

风格方向可偏向：

- 治愈感
- 电影旅行感
- 文艺纪念感
- 城市漫游感
- 轻户外感
- 自然日记感

---

### 5.6 校园 / 学习 / 日常记录 / 镜子自拍 / 桌面随拍

重点处理方向：

- 强调真实生活感与轻设计感
- 适合做成“随手拍升级版”
- 可加入笔记风、小标签、箭头、小手账、手绘边注
- 避免过于商业化，保留松弛和真实

风格方向可偏向：

- 清新校园
- 手账感
- 轻海报化
- 日常记录感
- 轻松可爱
- 少量杂乱拼贴感

---

### 5.7 夜景 / 演出 / 霓虹 / 灯光环境图

重点处理方向：

- 保留灯光氛围与夜色层次
- 修图重点放在压噪、控高光、提主体
- 涂鸦元素可以更大胆、发光、赛博、流动
- 字体可更锋利或更有视觉冲击力，但仍需与照片协调

风格方向可偏向：

- 夜游感
- 演唱会回忆感
- 赛博都市感
- 潮流街头感
- 迷离梦境感

---

### 5.8 静物 / 礼物 / 物件记录 / 产品感照片

重点处理方向：

- 让主体更有“陈列感”和“主视觉感”
- 涂鸦可做点题、解释、陪衬和气氛增强
- 适合加入贴纸感说明、小标签、小图标、轮廓描边、亮点强调

---

## 6. 风格自动匹配逻辑

请根据原图内容自动判断适合的整体方向。例如但不限于：

- 儿童主题：更童趣、活泼、跳跃
- 少女主题：更轻盈、清新、精致
- 少年主题：更干净、酷感、松弛
- 温馨日常：更自然、柔和、生活化
- 街头主题：更大胆、夸张、喷涂感更强
- 旅行主题：更自由、通透、带纪念感
- 宠物主题：更可爱、更拟人、更有互动趣味
- 美食主题：更精致、诱人、克制而有点睛感
- 夜景主题：更迷离、发光、氛围化
- 高级质感主题：更克制、更统一、更有设计感

## 7. 文字生成要求

### 7.1 文字功能

文字不应固定死板，而应根据场景智能生成。可作为：

- 主标题
- 副标题
- 角落短句
- 边注
- 小型标签
- 拟声词
- 氛围短语
- 关系感文案
- 纪念感文案
- 搞怪吐槽文案

### 7.2 文字风格

文字要求：

- 少而有力
- 与画面气质匹配
- 不要大段堆砌
- 不要喧宾夺主
- 可大可小，可集中也可分散
- 可理性、可温柔、可俏皮、可潮流、可文艺、可幽默

### 7.3 文字语言选项

修图与创意叠加后的文字内容，支持以下语言方向，由用户在最后自行选择；若用户未指定，则你可根据画面自动选择最合适的语言形式：

- 中文
- 英文
- 日文
- 韩文
- 中英混合
- 中日混合
- 中韩混合
- 英日混合
- 自动判断最合适语言

要求：

- 所选语言应与画面氛围一致
- 若使用混合语言，应自然，不要生硬拼接
- 若是潮流类、街头类、时尚类图，可适度使用英文短句
- 若是治愈、文艺、纪念类图，可优先简短且有情绪的文字
- 若用户提供指定文案，则优先使用用户文案
- 若未提供文案，则允许你智能生成合适内容

---

## 8. 字体与排版要求

字体风格不要固定为单一形式，应根据人物属性、场景类型、主题气质和整体美学自动匹配。

可选方向包括但不限于：

- 手写感
- 涂鸦感
- 蜡笔感
- 记号笔感
- 板绘字感
- 轻海报字体
- 杂志感字体
- 复古标题字
- 可爱圆体感
- 锋利街头字感
- 日记感小字
- 手账式标签字

排版要求：

- 有主次层级
- 疏密有变化
- 与涂鸦元素相互配合
- 不要满屏撒字
- 不要机械地居中排版
- 允许局部倾斜、手写边注、穿插式布局，但整体要协调

---

## 9. 色彩与氛围要求

色彩不要机械套用固定配色，而应根据以下内容智能匹配：

- 原图环境
- 光线状态
- 人物 / 主体属性
- 情绪走向
- 使用场景
- 所选创作风格

要求：

- 整体统一，有亮点，有记忆点
- 局部装饰色可适当跳出，但不能破坏和谐
- 如果原图色彩已经很好，可轻调，不要过度改色
- 如果需要提高视觉冲击力，可通过局部高饱和色、发光色、对比色、强调色来实现
- 不同主题要有不同色彩策略，例如：
  - 温馨日常：柔和暖色、低压感
  - 少女清新：浅彩、轻盈、透气
  - 街头潮流：高对比、跳色、喷漆感
  - 旅行风景：通透、空气感、自然高光
  - 夜景赛博：冷暖对冲、霓虹点缀
  - 美食甜品：诱人暖调、轻亮感

---

## 10. 互动元素要求

涂鸦元素必须与人物动作、手势、视线、表情和场景物件形成自然互动，不可只是贴在空白区域。

互动关系可包括但不限于：

- 围绕人物旋转
- 被人物托住 / 抱住 / 指向 / 注视
- 从人物动作中“冒出来”
- 顺着视线延伸
- 挂在场景物件上
- 与宠物 / 食物 / 风景发生关联
- 在人物周围形成情绪可视化

可以加入：

- 一个核心互动生物
- 若干小型配角
- 小物件
- 小角色
- 小贴纸
- 星星
- 爱心
- 闪电
- 涂抹块
- 动作线
- 喷漆云团
- 拟声词
- 表情元素
- 相机框
- 对话框
- 小标签
- 路径线
- 其他适合该图的创意语言

---

## 11. 装饰密度与完成度控制

请根据照片本身判断应采用哪种装饰密度：

### 11.1 轻度叠加
适合原图已很好看，只需轻点缀的情况。  
特点：简洁、干净、少量元素、强调高级感。

### 11.2 中度叠加
适合大多数朋友圈、日常记录、人像、美食、旅行图。  
特点：元素丰富但有节制，平衡真实感与创意感。

### 11.3 高密度叠加
适合潮流、街头、搞怪、拼贴、纪念海报类内容。  
特点：更强视觉冲击力，更丰富的图形语言，但仍需保持可读性和整体秩序。

---

## 12. 质量限制与避错要求

请严格避免以下问题：

- 过度修图，导致失真
- 涂鸦乱贴、没有互动逻辑
- 文字太多、太俗、太网感、太尬
- 字体与画面风格冲突
- 元素堆积无层次，导致脏乱
- 主体被挡住太多
- 美食被装饰压过
- 风景被过度填满，失去留白
- 人像五官失真、脸部不自然
- 用色杂乱、无统一性
- 强行可爱、强行文艺、强行高级
- 遮挡区域处理得很生硬
- 创意元素与场景无关
- 输出结果像廉价贴纸模板，而不是定制创作

---

## 13. 输出目标

请输出一张完成度高、构图完整、细节丰富、风格统一的成图，满足以下目标：

- 适合直接发布
- 一眼看出是“真实照片升级后的创意作品”
- 有互动感、有审美、有节奏
- 能体现照片内容本身的个性和情绪
- 不只是“修图”，也不只是“涂鸦”，而是两者融合后的完整创意表达
```
