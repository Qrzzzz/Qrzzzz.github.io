# 手绘教育信息图生成器

<p class="lead">这份提示词把一段知识性内容转换为单页手绘教育信息图，并约束纸张质感、配色、信息层级与图文比例。</p>

## 提示词简介

它用于把已经成形的知识内容压缩成一张单页视觉摘要，并根据内容关系自动选择流程、对比、循环、组成或层级布局。提示词强调图形优先、短标签和清晰的信息分区，适合呈现结构与要点，不适合原样承载大段文字。

### 适用情境

- 把课程笔记、科普内容或文章要点整理成便于理解和复习的知识图卡。
- 图解一个流程、机制、方法步骤或因果关系，让读者能快速看懂整体结构。
- 展示概念对比、分类体系、组成关系或层级结构，减少纯文字说明的负担。
- 为演示文稿、讲义、社交媒体或工作坊制作风格统一的单页视觉材料。

::: tip 使用方式
复制下方完整提示词，把末尾的 `<在此处插入内容>` 替换为需要图解的原文，再与具体尺寸、语言或必须保留的数据一同发送。
:::

```md
你是一位擅长手绘风信息图的视觉设计师。请根据以下内容创作一张单页信息图。

风格要求

整体风格：在温暖的奶油色纸张纹理（#F5F0E8）上绘制手绘教育信息图。所有线条和形状带轻微手绘抖动感，整体干净清晰，像高质量演示文稿的单页视觉摘要。无写实元素。

配色方案：
- 信息区色块：马卡龙色系圆角卡片——浅蓝 #A8D8EA、薄荷绿 #B5E5CF、薰衣草紫 #D5C6E0、浅桃 #F4C7AB，根据内容分区选用
- 强调色：珊瑚红 #E8655A，用于关键词、重要数据、勾选标记等需要视觉突出的元素
- 线条与主文字：黑色
- 辅助标注：暖灰 #6B6B6B，字号较小

图形优先：用图标、简笔画卡通形象、示意图承载信息，文字仅用于标注和点睛，能用图说清的绝不用文字。像优秀的演示文稿一样——一眼看懂结构，细看理解细节。

信息结构：根据内容自动选择最佳视觉布局（流程→箭头串联，对比→左右分栏，循环→环形，组成→并列卡片，层级→嵌套等）。用圆角色块、气泡、虚线框等容器分区，区域间用手绘波浪箭头连接并标注简短关系词。

文字层次：标题顶部居中，使用粗体大号手绘字；区域内用粗体关键词和暖灰小字短标签（2 至 5 个词）区分层次。

渲染细节：马卡龙色块不完全填满轮廓，加入涂鸦装饰点缀（小星星、下划线、小箭头等），保持充足留白和干净构图。

底部金句：图片底部一句粗体居中总结，概括核心观点。

要图解的内容
<在此处插入内容>
```
