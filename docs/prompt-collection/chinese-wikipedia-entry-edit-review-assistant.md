# 中文维基条目编辑与复核助手

<p class="lead">这组提示词将条目工作拆成编辑与独立复核两个阶段：第一段负责依据权威来源修订内容，第二段负责逐句、逐链接和逐项语法验收。</p>

## 提示词简介

两个阶段应当分开运行。编辑阶段专注于资料检索、事实修正、百科式改写和可直接粘贴的完整源代码，不提前承担最终验收；复核阶段则以编辑结果为输入，不擅自扩写或重组条目，只检查事实、引文、链接、站内目标、图片、模板和维基语法，并直接修正发现的问题。

### 适用情境

- 修订缺少来源、表述过度或事实已经过时的中文维基条目草稿。
- 整理翻译条目或多来源材料，需要统一中文维基格式、内部链接和参考文献模板。
- 在正式提交完整维基源代码前，希望由独立上下文逐句核对事实与引文，并实际检查全部链接。
- 需要把内容编辑和最终验收明确分工，分别保留编辑摘要、复核记录与最终编辑摘要。

::: tip 使用方式
先把待编辑的完整条目源代码附在第一段后运行，并保存其完整输出。随后开启新的对话或独立上下文，把编辑结果附在第二段后运行；若只需要其中一个阶段，也可以单独复制对应区块。
:::

## 编辑阶段

```md
请执行“编辑阶段”：查找并优先采用政府机构、学术机构、同行评审文献等权威来源，核实并修正不准确、无来源或表述过度的内容，可在可靠来源支持下略微扩充。保持中立、百科式语气和中文维基常用格式，避免原创研究；处理好内部链接、引文页码和参考文献模板。请输出可直接粘贴的完整维基源代码，并另给一条简洁的编辑摘要。本阶段只负责编辑，不进行最终链接可访问性或全文复核。

待编辑内容如下：

<在此处粘贴待编辑的完整维基源代码与必要背景>
```

## 复核阶段

```md
请执行独立的“复核阶段”，不要擅自扩写或改变条目结构：逐句核对事实与引文是否匹配，检查来源机构的权威性、引用页码、日期、标题和作者信息；实际访问全部外部链接并报告状态，检查图片、中文站内链接等目标是否存在，同时检查维基语法、模板和参考文献能否正确解析。发现问题时直接修正并输出复核后的完整源代码；另外列出修正项目、链接检查结果，并给出最终编辑摘要。若无问题，也要明确说明已经检查的项目。
```
