# 为什么要有这个网站

<p class="lead">这个网站用于长期整理和保存文档、项目记录及个人写作，让分散的内容拥有稳定、可以继续修订的入口。</p>

## 内容如何组织

与依赖时间排序和平台分发的信息流不同，独立网站可以按照内容本身的关系进行组织。文章发布后仍可持续修改，也便于日后检索和引用。

网站不会以更新频率或栏目数量为目标。内容经过整理后再发布，并根据实际情况持续修订。

## 更新方式

正文使用 Markdown 编写，与网站源码一同存放在 GitHub。内容推送至 `main` 分支后，GitHub Actions 会自动完成构建和发布。
